Professional and standard features

testIDEA has an advanced set of features, which are available only in professional mode. This mode is activated, if there is a license present in your emulator. To activate these features, emulator must be connected to your computer and turned on. Then you should connect to winIDEA with command iTools | Connect to winIDEA. If you connect or turn on the emulator later, simply execute the command again and testIDEA will check the license.

List of professional features:

  • Report generation
  • Wizard for test case generation
  • Test templates
  • I/O module & HIL support
  • Code coverage
  • Profiler/performance analysis
  • Trace
  • Python script generation
  • Script execution before/after test
  • Excel/CSV/CTE XL import/export
  • Filtering of tests for execution
  • Test specification in C/C++ sources
  • Dry run
  • Measurement of stack usage
  • Function execution sequence verification
  • Test points
  • Renaming of functions and variables
  • Native (real-time) stubs
