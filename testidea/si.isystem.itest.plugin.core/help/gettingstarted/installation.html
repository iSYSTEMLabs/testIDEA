Installation

This section describes how to install iSYSTEM testIDEA. It applies only for those who want to install testIDEA separately from winIDEA. If you have installed full winIDEA setup, testIDEA is already installed on your system, and you should skip this step.

Prerequisites

The following software needs to be installed on the system to be able to run and use iSYSTEM testIDEA:
  • Java Runtime Environment (JRE) version 1.7 is a must to run iSYSTEM testIDEA
  • winIDEA version 9.12 or later. This is not required to run iSYSTEM testIDEA, but only to run tests. You can write, edit, and save test specifications without winIDEA. However, content assistance (list of functions, variables, ... on input fields) is not available without winIDEA.

Download and install

testIDEA installation is part of winIDEA installation. If you need to update or install it separately from winIDEA, contact iSYSTEM support for the installation file. Then unzip the file to your hard disk. There is a iSystem_testIDEA.exe executable in the unzipped folder. Make desktop shortcut for your convenience.
