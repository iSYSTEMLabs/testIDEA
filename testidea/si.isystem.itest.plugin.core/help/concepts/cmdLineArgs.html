Command line arguments

iSYSTEM testIDEA understands the following command-line arguments:

-isys_ws <winIDEA_workspace> This option connects to winIDEA with the specified workspace opened. If no such winIDEA is running, one is started.
-isys_open <testIDEA_iyaml_file> This option loads the given YAML file at startup. Use this option when setting double-click action for *.iyaml files in Windows Explorer.
