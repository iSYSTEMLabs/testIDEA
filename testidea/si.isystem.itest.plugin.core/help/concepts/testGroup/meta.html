Meta

This section contains information, which describes group:
  • ID - group ID. testIDEA does not require group ID to be specified, but it is highly recommended to use this feature, because it makes possible to link results in test reports to groups.

  • Description - human readable description of the group.

  • Result comment - This text refers to specific test run. It is stored to results and report only, and will be lost on next test run.


See tool-tips for additional details and examples.
